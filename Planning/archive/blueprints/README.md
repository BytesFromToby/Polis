# Archived blueprints

Build plans for features that have shipped and been inspector-signed. Blueprints are build-time
scaffolding — once their feature is live, the spec is the truth and the blueprint is history (it
keeps the checkbox trail, deviation notes, and inspector stamps from that build).

`../../blueprints/` holds only the **current / most-recent** build; everything retired moves here.
Non-authoritative.
