================================================================================
UNIT
================================================================================

field               type / notes
---------------------------------------------------------------------------
id                  string — unique identifier
name                string
faction_1           FactionSlot | null  (primary — can lead here)
faction_2           FactionSlot | null  (secondary)
faction_3           FactionSlot | null  (tertiary)
is_leader           bool — cached. true if leading any faction.
traits              array of trait_id
edge                int 1–19  (edge + grit = 20)
grit                int 1–19  (edge + grit = 20)
health              int 1–100
domains             array of DomainRating
focus_1             FocusSlot | null  (primary focus)
focus_2             FocusSlot | null  (secondary focus)
is_npc              bool  (default true)

DomainRating object:
  domain_id         string
  rating            float  e.g. 4.67  (floor used for resolution)
  entrench          float 0.00–1.00

FactionSlot object (per slot):
  faction_id        string
  inertia           float 0.0–cap  (see 02_Constants for caps)

FocusSlot object:
  target_id         string — unit_id or faction_id
  target_type       enum: unit | faction | domain | goal
  score             float 0.0–10.0
  source_trait      trait_id | null


================================================================================
FACTION
================================================================================

field               type / notes
---------------------------------------------------------------------------
id                  string
name                string
domain_primary      domain_id
rating              float — decimal e.g. 5.73  (floor used for resolution)
traits              array of trait_id
leader_id           unit_id | null
leadership_need     float 0.0–20.0
entrench            int 1–100
member_ids          array of unit_id
level_1_count       int — background level 1 members (no individual tracking)
relationships       array of FactionRelationship  (overrides domain default)

FactionRelationship object:
  faction_id        string
  trait             enum: Friend | Foe | Neutral

resolution order for same-domain targeting:
  1. check faction.relationships for specific entry — use if found
  2. fall back to domain.relationships[this_domain] — the intra-domain default


================================================================================
DOMAIN
================================================================================

field               type / notes
---------------------------------------------------------------------------
id                  string
name                string
cap                 int — total weight supported  (SM: used as Attention)
utilization         float — sum of unit logarithmic weights in domain
drift               float — natural Entrench change per cycle (+/-)
relationships       array of DomainRelationship

DomainRelationship object:
  domain_id         string
  trait             enum: Friend | Foe | Hide | Client | Neutral

note: SM domain identified by name, not a flag.
      capacity is derived from floor(faction.rating) — not stored on faction.
      intra-domain default = domain.relationships entry where domain_id == self


================================================================================
WORLD STATE
================================================================================

field               type / notes
---------------------------------------------------------------------------
cycle               int
chaos               array of {domain_id, level: int 0-10}
power_vacuums       array of {domain_id, cycles_remaining: int, origin_unit_id}
attention           float — SM domain utilization as Attention
attention_state     enum: baseline | elevated | crisis


================================================================================
CYCLE EVENT
================================================================================

field               type / notes
---------------------------------------------------------------------------
cycle               int — which cycle this occurred
actor               unit_id or faction_id
action              action_id
target              unit_id | faction_id | domain_id | null
domain_used         domain_id
domain_targeted     domain_id | null
points              int — action points spent
result              string — narrative description of outcome
dramatic            int — 0 = routine. higher = more dramatic. scale TBD.
